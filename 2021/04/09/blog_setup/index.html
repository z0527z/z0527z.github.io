使用Hexo基于github搭建博客

  • 安装Node.js和配置好Node.js环境

  • 安装git和配置好git环境

  • 新建一个账户名.github.io 的项目，并且需要勾选 Initialize this repository with a README

  • 安装Hexo，输入npm install hexo -g 安装完成后输入 hexo -v检查版本

  • 进入到博客文件夹，输入hexo init进行初始化 输入npm install安装所需要的组件

  • 新建一篇博客，在cmd执行命令：hexo new post 博客名，这时候在文件夹_posts目录下将会看到已经创建的文件

  • 将hexo与github pages联系起来。在 ~/.ssh 下生成新的rsa私钥跟公钥，输入以下内容：ssh-keygen -t rsa -C ''邮箱" ，然后将本地 id_rsa.pub中的内容拷贝复制到github上settings里的ssh keys里面

  • 配置成功后修改执行hexo init文件下的_config.yml文件，配置最下面的Deployment, 配置如下：

  • 博客内容写完后在发布之前需要安装一个扩展：npm install hexo-deployer-git --save，安装完之后使用命令 hexo d 就可以生成部署了。

  • 部署成功后访问地址：http://用户名.github.io 就可以了
